## What I Built

I created a small multi-page website consisting of:

- A main **index page** that acts as a homepage
- Individual **recipe pages**, each with consistent structure
- Navigation links between the homepage and recipe pages

Each recipe page includes:

- A heading with the recipe name
- An image of the finished dish
- A short description
- An unordered list of ingredients
- An ordered list of preparation steps
